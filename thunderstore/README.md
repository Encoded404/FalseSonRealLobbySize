# FalseSonRealLobbySize

A Risk of Rain 2 mod that adjusts False Son's display size on the character selection screen.

## Description

By default, False Son appears at 0.7x his actual in-game size on the character selection screen. This mod allows you to adjust that multiplier to see his true scale.

Based on [this project by TheTimeSweeper](https://github.com/TheTimeSweeper/WagaTamashiiWaMadaMoeteOru/tree/master/SillyFalseSon) - all credit goes to him!

## Configuration

After first run, a config file will be generated at `BepInEx/config/Encoded404.Encoded404.FalseSonRealLobbySize.cfg`

| Setting | Default | Description |
|---------|---------|-------------|
| CSS Size Multiplier | 1.0 | Multiplier for False Son's size on the character selection screen. Vanilla is 0.7x his in-game size. |

## Installation

### Using a Mod Manager (Recommended)
1. Install using r2modman or Thunderstore Mod Manager
2. Launch the game through the mod manager

### Manual Installation
1. Install [BepInEx](https://thunderstore.io/c/riskofrain2/p/bbepis/BepInExPack/)
2. Download and extract this mod
3. Place `FalseSonRealLobbySize.dll` in your `BepInEx/plugins` folder

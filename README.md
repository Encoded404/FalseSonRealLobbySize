# FalseSonRealLobbySize

A Risk of Rain 2 mod that uses BepInEx.

based on [this project by TheTimeSweeper](https://github.com/TheTimeSweeper/WagaTamashiiWaMadaMoeteOru/tree/master/SillyFalseSon)

all credit goes to him, this is simply some of his logic extracted into a sepperate mod

## Building

1. Ensure you have .NET SDK installed
2. Run `dotnet restore` to restore NuGet packages
3. Run `dotnet build` to build the project

## Installation

1. Build the project
2. Copy `bin/Debug/netstandard2.1/FalseSonRealLobbySize.dll` to your `BepInEx/plugins` folder
